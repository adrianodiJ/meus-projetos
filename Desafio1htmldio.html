O Que É ?

(Voltar)

Criado por Tim Berners-Lee nos anos 90, se trata da base de Toda a internet junto com CSS e JavasCript

Ao contrario de muitas outras linguagens, o HTML em si não é uma linguagem de Programação, mas sim de Marcação

sendo especificamnete utilizado para a estruturação de sites.

Podemos utiliza-lo por exemplo para, definir titulos, parágrafos, imagens, links tabelas e vídeos

Semântica

(Voltar)

A semântica do HTML é o princípio de usar as tags corretas para representar o significado real do conteúdo em uma página da web.

Em vez de usar tags genéricas apenas para deixar algo "bonito", o HTML semântico foca em descrever o papel de cada parte do conteúdo de forma clara e compreensível, tanto para os desenvolvedores quanto para os navegadores e ferramentas assistivas, como leitores de tela.

Além disso, com a semântica auxiliando os leitores de tela, faz com que diversas pesoas com deficiencias auiovisuais se beneficiem,dando mais acessibilidade e participação destas pessoas dentro da intwrnet

O Que São Tags

(Voltar)

As tags HTML são comandos usados para marcar e estruturar o conteúdo de uma página da web. Elas funcionam como blocos de construção que indicam ao navegador o que cada parte do conteúdo representa: um título, um parágrafo, uma imagem, um link, entre outros.

Cada tag geralmente vem em par, com uma tag de abertura (como <'p'>) e uma tag de fechamento (como <'/p'>), envolvendo o conteúdo que ela define. Por exemplo, <'h1'>Bem-vindo<'/h1'> indica que "Bem-vindo" é um título principal.<'/p'>

Em resumo, as tags HTML servem para organizar e dar significado ao conteúdo de um site, dizendo ao navegador como exibir cada parte da página.

Exemplos de Tags

(Voltar)
  • <"Header"> Informação invisivel, Titulo e estilos
  • <"Body"> Todo o Corpo do site
  • <"h1" a "H6"> Adiciona Titulos ao site
  • <"p"> Adiciona paragrafos para escrita
  • <"a href="link"> Adiciona Link
  • <"img src="link da Imagem"> Adiciona Imagem
  • <"ul"> Adiciona listas não ordenadas ao site

HTML Hoje

(Voltar)

O HTML também é essencial para criar sites que funcionam bem em celulares e tablets, o que chamamos de sites responsivos. Além disso, ele é utilizado até mesmo em aplicações web mais complexas, como sistemas de bancos, redes sociais e plataformas de ensino. Portanto, o HTML ainda é amplamente utilizado hoje como a estrutura principal da web, servindo tanto para sites simples quanto para projetos modernos e avançados. Mesmo com novas tecnologias surgindo, ele continua sendo indispensável.
